
# Livish Portfolio (Vite + React + TS + Tailwind)

A polished portfolio site based on your component, upgraded with:
- Vite + TypeScript project setup
- Tailwind CSS styling
- Icons (lucide-react) and small motion (framer-motion)
- Accessible keyboard shortcuts (1–6) to switch sections
- Downloadable resume link

## Quick Start

```bash
# 1) Install deps
npm install

# 2) Start dev server
npm run dev

# 3) Build for production
npm run build

# 4) Preview production build
npm run preview
```

Open the dev URL that Vite prints (usually http://localhost:5173).

## Files of Interest
- `src/components/HomePage.tsx` — your enhanced portfolio component
- `public/resume.pdf` — replace this with your actual resume
- `index.html` — base HTML
- `tailwind.config.js` + `postcss.config.js` — Tailwind setup
